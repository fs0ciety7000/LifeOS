## Resume

Le concept de "Second Brain" est une methodologie pour capturer, organiser et utiliser l'information de maniere systematique. Popularise par Tiago Forte.

## Concepts cles

### PARA Method
Organisation en 4 categories:
1. **Projects** - Objectifs a court terme avec deadline
2. **Areas** - Responsabilites continues
3. **Resources** - Sujets d'interet
4. **Archives** - Items inactifs

### CODE Framework
1. **Capture** - Sauvegarder ce qui resonne
2. **Organize** - Trier par actionabilite
3. **Distill** - Extraire l'essentiel
4. **Express** - Utiliser et partager

### Progressive Summarization
Technique de lecture active:
- Layer 1: Note originale
- Layer 2: Passages en gras
- Layer 3: Surlignage
- Layer 4: Mini-resume

### Zettelkasten
Methode de prise de notes atomiques:
- Une idee = une note
- Liens entre notes
- Emergence de connections

## Applications personnelles

J'utilise une combinaison de:
- **PARA** pour la structure
- **Zettelkasten** pour les notes atomiques
- **Daily notes** pour le journaling
- **Dataview** pour les vues dynamiques
